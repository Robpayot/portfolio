This is an update to the "Bokeh" shader by Martins Upitis that is included in the three.js examples. It uses the most current code v2.4
Blog Post
Blender Artist Forum Post Describing Usage

The shader does calculations on a unit to meter scale, the best effects are achieved on sizes much smaller than a typical three.js scene. For example the interior here is only 10x10 and the colored lights have a radius of 0.1. When using the shader in a larger scale scene manual settings work best

The blur is done with a nested loop that could probably be optimized into two passes
